# Landing Page: Soluciones Tecnológicas para Bolivia

[![Estado del Proyecto](https://img.shields.io/badge/Estado-En%20Desarrollo-yellow)](#) [![Licencia](https://img.shields.io/badge/Licencia-MIT-green)](#licencia)

## 🚀 Descripción

Este proyecto es una **landing page** desarrollada para promocionar servicios tecnológicos especializados para el mercado boliviano. La página presenta tres soluciones principales: una plataforma de gestión de chats y citas, un sistema de facturación electrónica y soluciones personalizadas de Inteligencia Artificial.

La página se construyó utilizando una plantilla obtenida del repositorio **[Awesome Landing Pages](https://awesome-landingpages.vercel.app)**, adaptándola completamente al contenido y objetivos de negocio específicos.

🔗 **Demo en vivo**: [Próximamente](#)

## ✨ Características Principales

*   **Diseño Responsivo**: Se ve perfecto en dispositivos móviles, tablets y computadoras.
*   **Enfoque en Conversiones**: Diseñada para guiar al visitante hacia la acción (solicitar demo, elegir un plan).
*   **Contenido Específico para Bolivia**: Incluye detalles sobre precios en bolivianos (Bs.), compatibilidad con el SIN y regulaciones locales.
*   **Secciones Claras**:
    *   Encabezado con propuesta de valor.
    *   Presentación de servicios.
    *   Tabla de precios sencilla y transparente.
    *   Llamados a la acción estratégicos.
*   **Tecnología Moderna**: Desarrollada con las mejores prácticas mencionadas en la fuente de la plantilla.

## 🛠️ Tecnologías Utilizadas

Esta landing page se construye sobre la base de **HTML, CSS y JavaScript puro**, utilizando **Tailwind CSS** para un desarrollo rápido y eficiente, tal como se describe en la fuente de las plantillas.

*   **HTML5**: Estructura semántica del contenido.
*   **Tailwind CSS**: Framework de utilidades para estilizado rápido y responsivo.
*   **JavaScript (Vanilla)**: Para interactividad básica (menús, formularios).
*   **Git**: Para un control de versiones potente y profesional.

## 🎯 Adaptaciones Realizadas

Esta plantilla fue personalizada para reflejar fielmente la propuesta de negocio:

*   **Contenido**: Todo el texto (encabezado, servicios, precios, blog) fue reemplazado por el contenido generado y ajustado para Bolivia.
*   **Estilo**: Se ajustaron colores, tipografía y componentes (como las tarjetas de precios) para alinearse con la marca.
*   **Componentes**:
    *   Se creó una sección de precios (`Pricing`) con tres planes claros (Básico, Empresarial, Corporativo).
    *   Se estructuraron las secciones de "Funciones Principales" y "Servicios".
    *   Se integró el párrafo de cabecera ajustado como el principal mensaje hero.
